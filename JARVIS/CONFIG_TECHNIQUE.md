# Technical Configuration

## Environment
- **OS**: Windows
- **IDE**: Android Studio
- **Build System**: Gradle (Kotlin DSL)
- **Primary Language**: Kotlin
- **UI Framework**: Jetpack Compose

## Key Dependencies
- `androidx.compose.material3`: For Material 3 components.
- `androidx.navigation:navigation-compose`: For navigation.
- `coil-compose`: For image loading (inferred from previous tasks).
- `androidx.lifecycle:lifecycle-viewmodel-compose`: For state management.

## Project Structure
- `app/src/main/java/com/example/hallisanthe/ui/screens/`: Contains all UI screens.
- `app/src/main/java/com/example/hallisanthe/ui/theme/`: Contains the theme and color definitions.
- `app/src/main/assets/stitch_screens/`: Reference HTML designs from Stitch.
